第一题 A：海湾的校旗

时间限制：C/C++ 1000MS，其他语言 2000MS
内存限制：C/C++ 256MB，其他语言 512MB
难度：中等
分数：100
出题人：pyk

描述

海湾的校旗一直处于损坏的状态，现在请你把旗子显示出来，设旗杆长度为 h ，旗子的长度为 x ，宽度为 y ，输入 h ， x ， y ，请你输出这面旗子。

输入描述

输入共一行三个正整数，分别为旗杆的高度 h ，旗子的长度 x ，宽度 y 。

输出描述

输出这面校旗的样貌。

样例输入 1 

10 6 4

样例输出 1 

******
*    *
*    *
******
*
*
*
*
*
*

样例输入 2 

7 3 2

样例输出 2 

***
***
*
*
*
*
*

说明

1 ≤ h,x,y ≤ 100
保证 h > y ， x > y

提示

循环嵌套

正解 

h,x,y=map(int,input().split())
# 输入三个数，并使用map函数将输入的字符串转换为整数，赋值给h,x,y
for i in range(0,h):
    # 遍历h次
    print('*',end='')
    # 打印*
    if i==0 or i==y-1:
        # 如果i为0或者i等于y-1
        for j in range(0,x-1):
            # 遍历x-1次
            print('*',end='')
            # 打印*
    elif i<y:
        # 如果i小于y
        for j in range(0,x-2):
            # 遍历x-2次
            print(' ',end='')
            # 打印空格
        print('*',end='')
        # 打印*
    print()

第一题 B：图书馆的电缆

时间限制：C/C++ 1000MS，其他语言 2000MS
内存限制：C/C++ 256MB，其他语言 512MB
难度：中等
分数：100
出题人：pyk

描述

图书馆的电缆修了两个月，卡池都更新三轮了，现在请你把图书馆和地下埋的电缆显示出来，设图书馆的长度为 x ，宽度为 y ，电缆长度为 l ，电缆位置与图书馆的左边缘距离为 d ，输入 x ， y ， l ， d ，请你输出图书馆和地下埋的电缆。

输入描述

输入共一行三个正整数，分别为图书馆的长度 x ，宽度 y ，电缆的长度 l ，电缆位置与图书馆的左边缘距离 d 。

输出描述

输出图书馆和地下埋的电缆的俯视图。

样例输入 1 

6 3 4 2

样例输出 1 

******
******
******
  |
  |
  |
  |

样例输入 2 

10 5 5 3

样例输出 2 

**********
**********
**********
**********
**********
   |
   |
   |
   |
   |

说明

1 ≤ x,y,l,d ≤ 100
保证 d <= x-1

提示

循环嵌套

正解 

x, y, l, d = map(int,input().split())
for i in range(0, y):  # 输出图书馆
    for j in range(0, x):
        print("*", end="")
    print()
for i in range(0, l):  # 输出电缆
    for j in range(0, d):  # 输出电缆左边距
        print(" ", end="")
    print("|")

第二题 A：肠粉价格计算器

时间限制：C/C++ 1000MS，其他语言 2000MS
内存限制：C/C++ 256MB，其他语言 512MB
难度：简单
分数：100
出题人：pyk

描述

你去到食堂排队准备吃肠粉，发现窗口前排的队很长，心想阿姨每天都要承担这么大的计算量，决定给他们写一个计算器。猪肉肠粉 🐖 和牛肉肠粉 🐂 的价格分别是 6 元和 9 元。
输入第一个数字 s ，表示学生点的是猪肉肠粉 🐖 还是牛肉肠粉 🐂 ，猪肉肠粉 🐖 用 0 表示，牛肉肠粉 🐂 用 1 表示；输入第二个数字 n ，表示学生加了几条肠粉，每加一条肠粉需要多支付 1 元；
输出一个数字，表示学生需要支付的金额。

输入描述

输入共一行两个非负整数，分别为 s 和 n 。

输出描述

输出共一行一个正整数，表示学生需要支付的金额。

样例输入 1 

0 2

样例输出 1 

8

样例输入 2 

1 1

样例输出 2 

10

说明

0 ≤ n ≤ 10000 # 话说应该没有人能加这么多吧

提示

输入-计算-输出

正解 

s,n=map(int,input().split())
print(6+s*3+n)

第二题 B：海湾的小卖部

时间限制：C/C++ 1000MS，其他语言 2000MS
内存限制：C/C++ 256MB，其他语言 512MB
难度：简单
分数：100
出题人：pyk

描述

小卖部是海湾最繁华的地方，这里有很多的商品，有各种零食、饮料、文具等等。但是早上在小卖部排很长很长的队总是令人苦恼。
现在请你制作一个计算器，帮助海湾的小卖部计算同学们在这里购买的商品总价。

假设只有饮料和面包，分别是

  • 饮料的价格是** 3 元/瓶**
  • 面包的价格是** 5 元/个**

输入一行包含两个整数 m , n ，分别表示饮料的数量和面包的数量。

输入描述

输入共一行两个整数 m , n ，分别表示饮料的数量和面包的数量。

输出描述

输出共一行一个整数，表示商品总价。

样例输入 1 

1 1

样例输出 1 

8

样例输入 2 

2 3

样例输出 2 

21

说明

0 ≤ m ≤ 10000 # 饮料数量
0 ≤ n ≤ 10000 # 面包数量
富哥v我两块面包

提示

输入-计算-输出

正解 

m,n=map(int,input().split())
